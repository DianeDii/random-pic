* 这是一个基于Java+oss实现的**随机**图片/壁纸接口。
* 它主要有以下几个分类:
  * [风景](http://8.131.228.245:8090/sc)
  * [背景](http://8.131.228.245:8090/bg)
  * [动漫](http://8.131.228.245:8090/cc)
  * [动物](http://8.131.228.245:8090/al)
  * [美女](http://8.131.228.245:8090/wm)
  * [美食](http://8.131.228.245:8090/fd)
  * [明星](http://8.131.228.245:8090/sr)
  * [汽车](http://8.131.228.245:8090/cr)
  * [人物](http://8.131.228.245:8090/pn)
  * [影视](http://8.131.228.245:8090/mv)
  * [游戏](http://8.131.228.245:8090/gm)
  * [宗教](http://8.131.228.245:8090/rg)
* 目前仅支持展示PC端壁纸,图源来自彼岸图网
* 由于图片宽高比各不相同，所以少数图片显示可能会有拉大/压缩.调用接口可自行处理。
